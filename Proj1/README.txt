 To run:
1a, 1b: compile with: gcc [pre.c or sort.c] -o [pre or sort]
   run: ./pre
        ./sort
When running pre, enter input on one line in the format: studentName GPA. Exit with ctrl-d
When running sort, no prompt will be given. Enter names immediatley, exit with ctrl-d.

2: compile with: gcc pipe.c -o pipe
   run: ./pipe

3: compile with gcc forkCommand.c -o fork
   run: ./fork [additional parameters to pass to ls]
or to run plain ls
   run: ./fork
